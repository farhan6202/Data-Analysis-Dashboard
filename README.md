# Data-Analysis-Dashboard
This Power BI project analyzes ski resort data across various countries and continents, uncovering insights about features such as child-friendliness, night skiing, summer skiing, and snow parks. It explores how these attributes influence resort pricing, popularity, and overall performance through interactive visualizations.
